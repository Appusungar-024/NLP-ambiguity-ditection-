                              API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

POST /api/analyze
  • Analyzes text for ambiguities
  • Returns: highlighted tokens, scores, explanations
  • Example: {"text": "sentence to analyze"}

GET /api/examples
  • Returns list of example sentences
  • Used for UI quick-load feature

GET /api/stats
  • Returns model status and info
  • Shows available analysis methods
